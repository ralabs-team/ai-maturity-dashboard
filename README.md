# AI Maturity Dashboard

Survey-based dashboard for measuring AI maturity across teams and organizations.

It helps turn survey responses into simple views of:

- adoption
- skills
- impact
- culture
- readiness

## What It Does

- shows available survey forms
- accepts response CSV uploads
- generates organization, team, and individual insights
- surfaces patterns across projects, departments, and seniority

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React
- TypeScript
- Vite
- Recharts
